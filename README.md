# Gui-EmployeeRecord

A graphical user interface program that uses Tkinter, messagebox, Frame, StringVar to display a window that will allow user to store employees records.
User can input employee name, department they are part of, either by manually inputting it or selecting from the dropdown list of already listed department and lastly user can also store whether the employee is full time or part-time.
The user can also edit employeem by clicking on the employee name in the visible listbox and click on the edit button which contains the edit_employee() function and will allow the user to edit the selected employee.
